# HNU 教学评价自动填写

适用于湖南大学教务系统（HNU 教学评价系统）的 Tampermonkey 自动评价脚本。

## 功能特性

* 在学生阶段评二级页面生成「激活自动评价」按钮
* 自动进入待评价课程
* 自动完成客观题选择
* 自动填写主观评价内容
* 自动提交评价
* 自动确认弹窗
* 自动处理二级页面中的全部待评价课程
* 支持校内访问与 WebVPN 访问

---

## 默认评价策略

### 客观题

脚本默认选择：

* 满意度题目：**非常满意**
* 推荐意愿题目：**非常愿意**

## 配置说明

脚本顶部可修改以下参数：

| 参数               | 默认值   | 说明        |
| ---------------- | ----- | --------- |
| suggestionText   | 无     | 主观评价内容    |
| satisfiedText    | 非常满意  | 满意度选择     |
| recommendText    | 非常愿意  | 推荐意愿选择    |
| subjectiveWaitMs | 45000 | 提交前等待时间   |
| detailWaitMs     | 5000  | 进入评价页等待时间 |

## 安装步骤

### 1. 安装 Tampermonkey

浏览器推荐：

* Google Chrome
* Microsoft Edge
* firefox

安装 Tampermonkey 扩展后继续下一步。

---

### 2. 导入脚本

打开 Tampermonkey 管理面板：

```text
Tampermonkey
→ 管理面板
→ 实用工具
→ 导入
```

选择：

```text
hnu_evaluation_auto.user.js
```

或者直接复制脚本内容新建脚本粘贴。

---

## 使用方法

### 从二级评价页面进入

进入：
```text
学生阶段评二级页面
```

点击「激活自动评价」按钮启动脚本。

---

### 启动脚本

页面右下角会出现蓝色悬浮按钮：

```text
激活自动评价
```

点击后脚本开始运行。

---

## 自动执行流程

脚本执行过程如下：

```text
点击激活自动评价
↓
进入待评价课程
↓
填写客观题
↓
填写主观评价
↓
等待45秒
↓
点击确认提交
↓
自动确认弹窗
↓
返回课程列表
↓
继续下一门课程
↓
全部完成
```

---

## 执行状态

按钮会显示不同状态：

| 状态     | 说明        |
| ------ | --------- |
| 激活自动评价 | 等待启动      |
| 执行中... | 脚本正在运行    |
| 已执行    | 执行完成      |
| 执行失败   | 执行过程中出现错误 |

---

## 常见问题

### 脚本失效
可能原因：
* 教务系统页面结构发生变化
* 浏览器更新导致 Tampermonkey 兼容性问题
* 网络问题导致资源加载失败
* 脚本与其他浏览器扩展冲突
* 浏览器安全设置阻止脚本执行


### 执行过程中停止

按 F12 打开控制台查看错误信息：

```text
Console
```

脚本会输出错误原因。
